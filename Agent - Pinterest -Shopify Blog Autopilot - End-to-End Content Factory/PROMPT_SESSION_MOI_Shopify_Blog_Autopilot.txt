Bạn là agent hỗ trợ dự án **Shopify Blog Autopilot – End-to-End Content Factory** (Pinterest → Topic → Research → Write → Images → Validate → Review → Publish).

NGỮ CẢNH ĐÃ LÀM:
- Part 1–5: Pipeline nội dung, audit & fix automation, article review, workflow fixes.
- **Part 6**: Thiết lập tự động chạy + **heartbeat**:
  - Executor chạy trên GitHub Actions mỗi 10 phút (publish.yml).
  - Local runner: scripts/run_local_executor.ps1 hoặc .sh ghi local_heartbeat.json; nếu heartbeat còn mới (≤15 phút), Actions sẽ SKIP executor để tránh chạy đôi.
  - Supervisor (supervisor.yml) chạy mỗi 30 phút: đọc queue (Sheets), pipeline health, WF_ENABLED/ALLOW_PUBLISH; nếu không ổn định thì disable pipeline + mở incident + notify; nếu ổn thì dispatch review hoặc publish.

FILE QUAN TRỌNG:
- NEXT_AFTER_PART6.md – Tóm tắt Part 6, việc tiếp theo, khi nào hoàn thành (chạy mượt).
- LOCAL_RUNNER_AND_WORKFLOW.md – Checklist local + GHA chạy mượt (repo root, heartbeat, tránh chạy đôi).
- RUNBOOK.md – Biến điều khiển, khi nào bật publish, xử lý incident.
- config/decision_log.json, config/DECISION_LOG_README.md – Reconcile layer; pipeline_v2/reconcile_decision.py (--last N hoặc RECONCILE_LAST_N).
- .github/workflows/publish.yml, supervisor.yml – Executor + check heartbeat; Orchestrator.
- Instruction Shopify Blog Automation - Github Actions.txt – Chi tiết bài toán và guardrails.

ĐÃ LÀM (Part 6 + Reconcile + chạy mượt):
- Runbook, PROMPT session mới, Decision log + reconcile_decision.py, LOCAL_RUNNER_AND_WORKFLOW.md, NEXT_AFTER_PART6 (khi nào hoàn thành).
